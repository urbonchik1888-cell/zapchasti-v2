📌 БЫСТРЫЙ СТАРТ:

🌐 РЕЖИМ 1: Просмотр без регистрации
   1. Откройте файл: index.html
   2. Просматривайте все каталоги
   3. Контакты продавцов скрыты
   4. Добавление товаров недоступно

🔐 РЕЖИМ 2: Полный доступ (с регистрацией)
   1. Откройте файл: auth.html
   2. Войдите как администратор:
      Логин:  admin
      Пароль: admin
   3. Или зарегистрируйте свой аккаунт
      (укажите телефон - будет виден другим)
   4. После входа доступны все функции

👥 РОЛИ ПОЛЬЗОВАТЕЛЕЙ:

👀 Гости (без регистрации):
   - Просматривают все каталоги
   - Используют поиск
   - НЕ ВИДЯТ контакты продавцов
   - НЕ могут добавлять объявления

👑 Администратор (admin):
   - Может удалять ЛЮБЫЕ объявления
   - Видит контакты всех продавцов
   - Видит точное количество всех товаров
   - Полный контроль над системой
   - Логин: admin / Пароль: admin

👤 Обычные пользователи:
   - Регистрируются с телефоном и email
   - ВИДЯТ контакты продавцов
   - Могут добавлять свои объявления
   - Могут РЕДАКТИРОВАТЬ только свои объявления
   - Могут УДАЛЯТЬ только свои объявления
   - Видят статус наличия новых запчастей (Есть/Нет)
   - Видят точное количество б/у товаров
   - Видят автора каждого объявления

⚠️ ВАЖНО:

• Это ЛОКАЛЬНАЯ система (работает на одном компьютере)
• Все данные хранятся в браузере
• Разные браузеры = разные базы данных
• Несколько человек могут работать по очереди
